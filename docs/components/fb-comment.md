Original Plugin Documentation of this feature found [here](https://developers.facebook.com/docs/plugins/comments)

## Props

#### url
The URL of the page which comment will be used.
> **Type**: *String*

> **Default**: *Current URL*

#### mobile

> **Type**: *Boolean*

> **Default**: *false*

#### numposts
The Number of Comments per page
> **Type**: *Number*

> **Default**: *10*

#### orderBy
> **Type**: *String*

> **Default**: *social*

#### colorsheme

> **Type**: *String*

> **Default**: *light*
